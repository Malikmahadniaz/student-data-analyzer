# Student Data Analyzer (C++)

A simple C++ project for data science beginners that reads student data from a file and performs basic statistical analysis.

## Features
- Reads data from file
- Calculates average, maximum, and minimum scores
- Counts pass and fail students
- Saves results to output file

## Concepts Used
- Structures
- File handling
- Arrays and loops
- Basic statistics

## How to Run
```

g++ main.cpp -o student-data-analyzer
./student-data-analyzer

```
